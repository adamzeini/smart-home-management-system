# Smart Home Management System

A Java console project for managing a simple smart home system.

The system includes rooms, lights, appliances, power limits, and day/night mode. It has two login modes: admin mode for managing the system, and user mode for controlling devices.

## Features

* Admin and user modes
* Add, delete, and search rooms
* Add, delete, and search lights and appliances
* Turn devices on and off
* Track power usage
* Low, normal, and high power modes
* Day and night mode
* Standby lists for devices that cannot run immediately
* Admin confirmation for critical devices

## Main Classes

* `Main`
* `ManagementSystem`
* `Device`
* `Light`
* `Appliance`
* `Room`
